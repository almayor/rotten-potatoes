 - Node.js
 - Express.js
 - Handlebars
 - Bootstrap
 - Heroku
 - AJAX
 - jQuery

---

To run the website
 
 1. `git clone https://github.com/almayor/rotten-potatoes`
 2. `cd rotten-potatoes`
 3. `npm install`
 4. `nodemon`
 5. Open `localhost:3000` in your browser. 

Entry points are at

- `/`
- `/admin`
